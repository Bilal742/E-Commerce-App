JavaScript – 40 Questions & Detailed Answers (Roman Urdu)

JavaScript – 40 Questions

1. JavaScript kya hai?
JavaScript ek programming language hai jo webpages ko interactive banati hai — buttons, forms, animations, API calls waghera. Browser me run hoti hai aur server-side bhi (Node.js) run kar sakti hai.
2. var, let aur const me kya farq hai?
`var` function scope rakhta hai aur hoist hota hai. `let` aur `const` block scope rakhte hain. `const` ki value reassign nahi hoti. Example: 
var a = 1;
let b = 2;
const c = 3;
3. JavaScript ke basic data types kaunse hain?
Primitive types: String, Number, Boolean, Null, Undefined, Symbol, BigInt. Non-primitive: Object (arrays, functions bhi objects hote hain).
4. Function declaration aur function expression me kya farq hai?
Function declaration hoist hoti hai (upri se call kar sakte ho). Function expression tabhi available hoti hai jab assign ho chuki ho. 
function foo(){ }        // declaration
const bar = function(){ } // expression
5. Arrow functions kya hoti hain aur kab use karte hain?
Arrow function shorthand syntax hai: `const f = (a)=>a*2`. Ye apna `this` nahi banati (lexical this), isliye callbacks aur method binding me useful hai. Lekin constructor ke liye use nahi karte.
6. Hoisting kya hota hai?
Hoisting JS ka behavior hai jisme declarations (functions and var variables) execution se pehle engine ne conceptually top pe le lete hain. `var` variables undefined se initialize hotay hain; `let/const` temporal dead zone me rahte hain jab tak declaration nahi hui.
7. Scope kya hota hai?
Scope batata hai ke variable kis area se accessible hai: Global scope (poori file), Function scope (`var`), Block scope (`let/const`).
8. Closure kya hai? Example do.
Closure ek function ko uske outer scope ke variables access karne ki ability deta hai, even jab outer function execute ho chuka ho. 
function outer(){
  let x = 5;
  return function(){ console.log(x); } // closure
}
const fn = outer(); fn(); // prints 5
9. What is the DOM?
DOM (Document Object Model) HTML ka programmatic representation hai. JS DOM API se elements select, create, modify, delete kar sakte ho (`document.getElementById`, `querySelector` etc.).
10. How to select elements in DOM?
Common methods: `document.getElementById('id')`, `document.getElementsByClassName('cls')`, `document.querySelector('.cls')`, `document.querySelectorAll('div')`.
11. Event bubbling vs capturing kya hai?
Event propagation ke do phases: capturing (parent -> child) aur bubbling (child -> parent). Default listener bubbling me run hota. `addEventListener(type, fn, {capture:true})` se capturing use kar sakte ho.
12. Event delegation kya hota hai?
Event delegation ek technique hai jisme parent element pe single listener lagaya jata hai aur event target check karke children handle kiye jate hain — performance aur memory ke liye acha hai.
13. What is `this` in JavaScript?
`this` context depend karta hai ki function kis tarah call hua: global call me window/undefined, object method me object, constructor me new instance, arrow functions me lexical `this` (outer lexical context se lega).
14. Difference between == and ===?
`==` loose equality hai (type coercion karta hai), `===` strict equality hai (type + value dono compare karta hai). Prefer `===` for predictable results.
15. What is NaN? Kaise check karte ho?
NaN = Not-a-Number, numeric operation fail hone pe aata hai. `isNaN(value)` ya better `Number.isNaN(value)` se check karo. Note: `typeof NaN` = "number".
16. What is prototype?
Prototype ek object hota hai jisse objects inherit karte hain. JS me functions ke `prototype` property se shared methods define karte hain; objects ke `__proto__` se lookup hota hai.
17. What is prototypal inheritance?
Objects ek dusre se directly inherit karte hain through prototype chain. Agar property object pe nahi mili to JS prototype chain check karta hai.
18. What is `class` in ES6 and how is it related to prototype?
`class` ES6 syntax sugar hai constructor functions + prototype ke upar. Example: 
class Person {
  constructor(name){ this.name = name; }
  greet(){ console.log('Hi', this.name); }
}
Under the hood methods `Person.prototype` pe add hoti hain.
19. What is an IIFE?
IIFE = Immediately Invoked Function Expression — function jo turant execute hoti hai, mostly scope isolation ke liye: 
(function(){ /* code */ })();
20. What is callback function?
Callback function wo function hota hai jo dusre function ko argument ke roop me diya jata hai aur future me call hota hai (e.g., event handlers, async operations).
21. What is Promise?
Promise ek object hai jo async operation ka future result represent karta hai. States: pending, fulfilled, rejected. Use `.then()` and `.catch()` to handle results.
22. What is async/await?
`async` function return karta hai Promise. `await` promise resolution ka wait karta hai inside async function, code synchronous jaisa likhne me madad karta hai. 
async function getData(){
  const res = await fetch('/api');
  const data = await res.json();
}
23. Difference between callback, promise and async/await?
Callback: nested callbacks lead to callback hell. Promise: better chaining and error handling. Async/await: syntax sugar over promises, easier readable code.
24. How to handle errors in async code?
Promises: `.catch(err=>...)`. Async/await: use `try { await ... } catch(err){ ... }` to catch errors.
25. What is fetch API and how to use it?
`fetch()` browser API for network requests. Returns a Promise. Example: 
fetch('/api/data')
  .then(res => res.json())
  .then(data => console.log(data))
  .catch(err => console.error(err));
26. What is JSON and how to parse/stringify?
JSON (JavaScript Object Notation) ek data exchange format. `JSON.parse(jsonString)` to convert to object, `JSON.stringify(obj)` to convert to string.
27. What is localStorage and sessionStorage?
Browser storage APIs: `localStorage` data permanent store karta hai (until cleared). `sessionStorage` tab tak rehta hai jab tak tab session open ho. Use `localStorage.setItem('k','v')` and `getItem`.
28. What is event loop?
Event loop JS runtime mechanism hai jo call stack aur task queue ko manage karta hai, async callbacks ko schedule karta hai. Ye single-threaded behavior me non-blocking IO allow karta hai.
29. What are microtasks and macrotasks?
Microtasks (Promises `.then`) high priority queue; macrotasks (setTimeout, I/O). Event loop pe microtasks run hote hain before rendering and before next macrotask.
30. How to debounce and throttle?
Debounce: function ko last call ke baad certain time pe run karna (useful for input). Throttle: function fixed interval me maximum ek baar run kare (useful for scroll). Example simple debounce: 
function debounce(fn, delay){
  let timer;
  return function(...args){
    clearTimeout(timer);
    timer = setTimeout(()=>fn.apply(this,args), delay);
  }
}
